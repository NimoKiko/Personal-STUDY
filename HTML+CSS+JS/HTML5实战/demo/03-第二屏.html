    星守千年

    以世间的花,蛇为灵感,以青色为主，更能够突出清姬的淡雅与幽静。蛇的受伤如同清姬心中的伤痕。

    这世间的百花,总能让清姬回顾之前与他共度的美好时光。还有那如影随形的头骨，仿佛他从未离开过一般。如果有人想抢夺她手中的头骨，她会毫不留情的让他从这世上消失，挫骨扬灰。
